# Choose storage solutions

Where and how you store research data affects security, accessibility, and compliance. Make storage decisions before data collection begins.

## Key considerations

- **Sensitivity:** Does the data contain personal identifiers or politically sensitive information?
- **Access:** Who needs to work with the data, and from where?
- **Volume:** How large will the dataset be?
- **Regulations:** Does Norwegian law or a funder require storage in a specific jurisdiction?

## Storage options

| Option | Best for | Notes |
|--------|----------|-------|
| CMI network drives | Day-to-day project files | Backed up, access-controlled |
| Cloud storage (OneDrive/SharePoint) | Collaboration with external partners | Check data sensitivity level first |
| Encrypted local storage | Highly sensitive fieldwork data | Use when connectivity is limited |
| Institutional archive (NSD/Sikt) | Long-term preservation | Typically used after project completion |

!!! tip "Fieldwork in low-connectivity settings"
    If you will be collecting data in areas with limited internet access, plan for encrypted local storage and a synchronisation strategy for when connectivity is available.
